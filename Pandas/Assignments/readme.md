## 🧩 PART 4 — Feature Engineering

15. Content Age
    From `release_year`
    Create column: content_age = current_year - release_year

16. Number of Genres
    `listed_in` has multiple genres separated by commas
    Count number of genres per show

17. Primary Genre
    Extract first genre from `listed_in`

18. Duration Number
    From `duration` column
    Example:
    "90 min" → 90
    "2 Seasons" → 2
    Create numeric column

19. Content Type Flag
    Movie = 0, TV Show = 1

20. Actor Count
    From `cast` column
    Count number of actors per show
